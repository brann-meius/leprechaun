# Leprechaun Project

### Requirements

- Docker
- Docker Compose

## Quick Start

1. Clone the project:
    ```bash
    git clone git@github.com:brann-meius/leprechaun.git leprechaun
    ```
2. Run:
    ```bash
    cd leprechaun
    ```
3. Run:
    ```bash
    make init
    ```
4. Welcome to:
    ```http
    http://localhost
    ```
